## The "Sticker Effect" - Our Signature

Most websites use this shadow pattern:
```css
box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
```

**KadaiPOS uses this unique pattern:**
```css
box-shadow: 4px 4px 0 #FF5A5F, 8px 8px 0 #1A1A2E;
```

This creates a **physical, tactile feeling** like actual stickers - something you can almost touch. Nobody else does this in the POS industry.
